# Meta Ads — Bella Imagem

Pasta de trabalho para produção de criativos e campanhas Meta (Facebook/Instagram) do projeto Bella Imagem.

## Estrutura

```
meta-ads/
├── README.md                     ← você está aqui
├── 01-briefing.md                ← entendimento do negócio e objetivos
├── 02-benchmark.md                ← análise de concorrentes (Meta Ads Library)
├── 03-estrategia.md               ← estrutura de campanha, públicos, budget
├── 04-copies.csv                  ← variações de headline, texto e CTA
├── 05-criativos/                  ← arte final por formato
│   └── README.md                  ← nomenclatura e organização
├── 06-checklist-publicacao.md     ← conferência antes de publicar
└── 07-relatorio-template.md       ← modelo de relatório diário/semanal
```

## Fluxo de trabalho (openclaw)

1. **Preencher** `01-briefing.md` com o cliente antes de qualquer outra coisa
2. **Validar briefing** → só então começar `02-benchmark.md`
3. **Apresentar estratégia** em `03-estrategia.md` para aprovação
4. **Produzir copies** (`04-copies.csv`) e **criativos** (`05-criativos/`) em paralelo
5. **Conferir checklist** (`06-checklist-publicacao.md`) antes de subir campanha
6. **Acompanhar** resultados em `07-relatorio-template.md`

## Regras gerais

- **Uma etapa por vez**: não avançar sem validação do gestor
- **Versionar**: nunca sobrescrever variações — numerar (`_v1`, `_v2`)
- **Nomenclatura padrão**: `BELLA_[FORMATO]_[ANGULO]_[VERSAO]`
- **Idioma**: copies sempre em pt-BR, tom alinhado ao briefing
- **Fonte de verdade**: este diretório — decisões verbais precisam ser registradas aqui
